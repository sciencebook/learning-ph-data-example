# Quick Start

Simply import this repository into Google Colab for easiest usage!

Otherwise, to run locally:

## Setup
Start virtual environment and install dependencies

```bash
# Create python virtual environment
python3 -m venv .venv

# Make the main file executable
chmod +x main.py

# Start venv
source .venv/bin/activate  # Linux/Mac
# .venv\Scripts\activate   # Windows

# Install dependencies (need to do once only)
pip install -r requirements.txt
```

## Run
```bash
# Start venv
source .venv/bin/activate  # Linux/Mac
# .venv\Scripts\activate   # Windows

# Run it
./main.py
```
